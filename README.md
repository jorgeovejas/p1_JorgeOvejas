# Componentes y lógicas:

- App.tsx:
Es la encargada de la lógica de llamada a la API así como de manejar excepciones de errores en la llamada a esta.


- CharacterList:
Encargado de mapear el array de personajes, a continuación se conecta con CharacterCard para crear una tarjeta

- CharacterCard:
Recibe la información del Personaje y la organiza en una tarjeta con las propiedades a la que esto haga referencia

- Loader:
Encargado de generar el icono de cargando y que sea funcional

- ErrorMessage:
Muestra de mensajes de error
